# تصميم واجهة تطبيق Mrium MEXC Trial

## نظرة عامة

تطبيق تداول العملات الرقمية لمنصة MEXC يعمل على Android 15 بتصميم داكن احترافي. التطبيق يدعم الوضع التلقائي السحابي 24/7 والوضع اليدوي.

## قائمة الشاشات

### 1. الشاشة الرئيسية (Home)
**المحتوى:**
- رأس الصفحة: صورة المستخدم الدائرية + اسم التطبيق + حالة البوت (ON/OFF toggle)
- مؤشر اتصال MEXC API (أيقونة واي فاي + "متصل" باللون الأخضر)
- زوج التداول BTC/USDT مع السعر الحالي وتغيير 24 ساعة
- رسم بياني شموعي Candlestick مع فترات زمنية (1د، 15، 55، 1س، 4س، 1ي)
- زرا الوضع التلقائي السحابي والوضع اليدوي
- مفتاح وصول API (مخفي) + عنوان IP المصرح به
- إجمالي الاستثمار (USDT) + المبلغ لكل صفقة (+/-) + محفظة العقود الآجلة (% ربح اليوم)
- شريط أخبار السوق (LIVE)
- بطاقات أسواق: النفط الخام، المعادن، الأسهم
- قسم المكافآت مع عداد تنازلي 10 دقائق

### 2. المحفظة (Portfolio)
**المحتوى:**
- الرصيد الإجمالي USDT
- ربح/خسارة اليوم مع النسبة المئوية
- قائمة الصفقات المفتوحة
- إدارة المخاطر: الهامش الأولي، هامش الصيانة، نسبة المخاطرة/العائد
- محفظة العقود الآجلة مع تفاصيل كل مركز

### 3. الصفقات (Trades)
**المحتوى:**
- قائمة الصفقات النشطة (فتح/إغلاق)
- سجل الصفقات المغلقة
- إحصائيات: إجمالي الصفقات المفتوحة، المبالغ، الأرباح
- فلترة حسب نوع العقد والزوج

### 4. التقارير (Reports)
**المحتوى:**
- ملخص الأداء اليومي والأسبوعي والشهري
- رسوم بيانية للأداء
- تفاصيل المكافآت المحصودة
- تقارير المخاطر

### 5. الإعدادات (Settings)
**المحتوى:**
- إعدادات API (إدخال وتشفير المفاتيح)
- إعدادات السحابة التلقائية
- إعدادات الإشعارات
- إعدادات الأمان ومشرف المحفظة
- نسخة احتياطية واستعادة

## التدفقات الرئيسية للمستخدم

### تدفق الاتصال
1. المستخدم يفتح التطبيق → يرى حالة الاتصال
2. يضغط على مفتاح API → يدخل المفاتيح → يتم تشفيرها
3. حالة الاتصال تتغير (أحمر/أخضر)

### تدفق التداول اليدوي
1. التبديل إلى الوضع اليدوي
2. إدخال المبلغ والرافعة
3. تنفيذ الصفقة
4. مراقبة النتيجة

### تدفق السحابة التلقائية
1. تفعيل الوضع التلقائي
2. التطبيق يعمل 24/7 في الخلفية
3. يجمع المكافآت كل 10 دقائق
4. يراقب الأخبار ويعدّل الصفقات
